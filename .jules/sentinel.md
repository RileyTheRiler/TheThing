## 2024-12-31 - Path Traversal Vulnerability in Persistence System\n**Vulnerability:** The save system allows user-controlled filenames without sanitization, leading to path traversal.\n**Learning:** Input validation must occur at the lowest level of file I/O operations to prevent directory traversal.\n**Prevention:** Sanitize filenames to allow only alphanumeric characters and safe separators.
